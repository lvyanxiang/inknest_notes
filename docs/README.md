# InkNest Notes Docs

This directory is organized by purpose so project execution, study notes, and
graduation materials do not crowd the same folder.

## Structure

- `development/`: project status, roadmaps, storage notes, subscription planning,
  and feature planning.
- `learning/flutter/`: Flutter study notes and keyword references.
- `academic/`: graduation task book, opening report, and future academic
  deliverables.

## Quick Links

- Current project status: `development/STATUS.md`
- MVP roadmap: `development/ROADMAP.md`
- Post-MVP roadmap: `development/POST_MVP_ROADMAP.md`
- Graduation task book: `academic/GRADUATION_TASK_BOOK.md`
- Opening report: `academic/OPENING_REPORT.md`
- Flutter keywords: `learning/flutter/FLUTTER_KEYWORDS.md`

When a roadmap or project-scope decision changes, update both the relevant
development document and the academic documents if the graduation scope is
affected.
