# Development Docs

Project execution documents live here.

- `STATUS.md`: short resume point for the next session.
- `ROADMAP.md`: MVP milestones and paused sync/backup scope.
- `POST_MVP_ROADMAP.md`: post-MVP feature planning.
- `STORAGE.md`: local file storage layout.
- `SMART_INK_PLAN.md`: Smart Ink product and technical plan.
- `SUBSCRIPTION_PLAN.md`: future packaging and monetization plan.

New or materially changed requirements enter through `$inknest-project`, which
routes product analysis to `$inknest-product-manager` and visible interaction
design to `$inknest-ui-ux` before implementation. Accepted durable decisions
are recorded under `docs/product/` and `docs/design/`.
